==========================================================
INSTALLATION
==========================================================

[Etape 1]

Il suffit d'éditer le fichier " infos.php " à l'aide d'un éditeur texte . 
Tout y est expliqué ce n'est pas très compliqué.

[Etape 2]

Une fois le infos.php édité, il ne reste qu'a uploader le contenu du dossier 
aiftepay 1.0 sur votre ftp.

[Mode d'utilisation]

Pour ajouter des fichiers sur aiftepay, il suffit de les uploader dans le dossier 
renseigné dans le fichier infos.php (par default il sagit du dossier " content " )

ATTENTION : Vos noms de dossiers et de fichiers ne doivent pas contenir le caractère | , 
ils ne seront pas accessibles autrement.
